# ManifestReel AI — Project Instructions

## Overview
AI-powered manifestation/spiritual reel creation platform. Users enter prompts, select style/platform/voice/mood, and get generated reels with scripts, captions, hashtags, and descriptions.

## Tech Stack
- Next.js 14, React 18, Tailwind CSS 3, Framer Motion
- PostgreSQL with Prisma ORM
- NextAuth.js v4 (credentials provider)
- Recharts for admin data visualization

## Design System
- **Theme**: Premium dark mode, forced dark
- **Background**: #0A0A0A (deep black)
- **Primary accent**: #D4AF37 (gold) — buttons, highlights, CTAs
- **Secondary accent**: #7B2FBE to #4A1A8A (purple gradient) — cards, tags
- **Typography**: DM Sans (body), Plus Jakarta Sans (display), JetBrains Mono (data)
- **Style**: Calm/Headspace-inspired, spiritual luxury aesthetic
- **Effects**: gold-glow, purple-glow, animate-float, gold-gradient classes

## Architecture
- **Provider Pattern**: Every AI service (Script, Voice, Music, Video, Render) follows Provider<TInput, TOutput> interface with mock + real implementations
- **Factory Functions**: Check env vars for API keys → return real or mock provider
- **Quota Enforcement**: Server-side only. Free=1 lifetime, Pro=30/mo, Premium=60/mo
- **Watermark**: Server-side flag based on tier, never trusted from client
- **Cost Tracking**: Every reel stores cost_breakdown jsonb + totalCost decimal
- **Generation Pipeline**: Async fire-and-forget with polling (GET /api/jobs/[id] every 2s)

## Database Tables
users, subscriptions, usage_counters, credits, reels, generation_jobs, social_accounts, assets

## Pages
- / — Landing page (hero, features, pricing, testimonials)
- /login, /signup — Auth pages
- /dashboard — Create reel form + recent reels
- /dashboard/generate/[jobId] — Live progress screen
- /dashboard/reel/[reelId] — Preview & edit (script, caption, hashtags, description)
- /dashboard/reel/[reelId]/schedule — Schedule/download flow
- /dashboard/library — Reel library grid with search/filter
- /dashboard/settings — Account, subscription, API status
- /admin — Admin overview stats
- /admin/reels — All reels with cost data
- /admin/users — User management
- /admin/margins — Revenue vs cost chart (recharts)
