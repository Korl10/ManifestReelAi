

## Voice Library + Custom Voice + Landing Hover Videos (added this session)

### Landing hero hover-play videos
- `app/_components/landing-page.tsx` SHOWCASE entries each have a `video` field: `/showcase/wealth.mp4`, `/showcase/selflove.mp4`, `/showcase/dream.mp4` (clean, no-text, muted, looping 540x960 Ken Burns clips built via ffmpeg from CDN backgrounds).
- `ShowcaseReel` component plays video on mouseenter/touchstart, pauses+resets on mouseleave. "Hover to play" badge hides on hover.

### Expanded voice library (dashboard "Pick a Voice")
- `app/dashboard/page.tsx`: `VOICE_LIBRARY` = 4 categories (Female 6, Male 6, Meditation 3, Motivational 3) of preset voices. Derived `ALL_PRESET_VOICES` + `VOICE_CATEGORIES`.
- 18 real samples generated via edge-tts in `public/voices/library/*.mp3`. Category cover images at `public/voices/{female,male,meditation,motivational}.jpg` (reused as card bg for all variations in a category — intentional, not a real duplicate).
- UI: category tab row + "My Voices" tab. Active category shows horizontal variation cards (play/pause preview, gold border when selected). All `<audio>` rendered once in hidden block keyed by voice id so playback survives tab switches.
- Generate payload now sends `voice` as the voice id string (e.g. `female-aria`, or custom voice id). Pipeline treats voice as free-string label — no validation needed.

### Add Your Own Voice (record/upload)
- Cloud storage initialized (S3). `lib/aws-config.ts` + `lib/s3.ts` (presigned upload w/ ContentDisposition=attachment for public files — client MUST send `Content-Disposition: attachment` header on PUT).
- Prisma `CustomVoice` model (userId, name, cloudStoragePath, isPublic, durationSec, source, @@map custom_voices) + `customVoices` relation on User.
- API: `app/api/voices/route.ts` (GET list w/ playable URLs via getFileUrl, POST save record), `app/api/voices/presign/route.ts` (POST presigned URL), `app/api/voices/[id]/route.ts` (DELETE w/ ownership check + S3 delete).
- `components/add-voice-modal.tsx`: record (MediaRecorder, 30s max, audio/webm) OR upload (audio/*, 25MB max) with preview. onAdded returns `{id,name,source,audio}`.
- "My Voices" tab shows custom voice cards (play + delete) + dashed "Add Your Own Voice" card opening the modal.
