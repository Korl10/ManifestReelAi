## Pipeline / infra
- `lib/generation-pipeline.ts` orchestrates: script → images → (motion) → voice → subtitles → music → composite.
- Image gen: Abacus image-gen.ts (gpt-5.4, 1024x1536) + Flux (fal) for motion tiers.
- Voice: ElevenLabs (lib/voice-catalog.ts 57 voices; tiers flash/multilingual/turbo). components/voice-browser.tsx, subtitle-editor.tsx (dynamic ssr:false).
- Compositor: FFmpeg via Abacus API, filter_complex MUST be double-quoted, 1080x1920@30fps.
- fal.ai content policy: avoid divine/sacred/spiritual/temple/god/angel (422). Motion uses camera-only templates (motion-prompts.ts). Flux uses enable_safety_checker:false + safety_tolerance 5.
- .env: STRIPE_SECRET_KEY, FAL_KEY (key_id:key_secret), ELEVENLABS_API_KEY, ABACUSAI_API_KEY.
- Run tsx: `cd nextjs_space && set -a && . ./.env && set +a && node_modules/.bin/tsx file.ts`
