## Reel duration pipeline (CRITICAL root-cause knowledge)
- `scriptPlanForDuration` in `lib/providers/llm-script.ts` previously had a hard floor of `Math.max(10, ...)` â†’ 16 words / 4 scenes min â†’ ~8s narration even for short reels. THAT was the bug that forced the old 5s reels to render ~8s and made the duration check unreliable.
- FIX: short-reel branch â€” `if (target <= 9) { wordBudget = max(9, round((target-2.0)/0.55)); return { affirmationLines: 2, totalScenes: 3, wordBudget }; }`. Narration lands UNDER target; pipeline then EXTENDS the final scene to exactly the target â†’ delta=0.
- `lib/generation-pipeline.ts` duration check (~line 625): Â±1s gate is ENABLED for ALL tiers now (removed the old free-tier skip). Verified delta=0 for 7s free reels.

## Thumbnails (real first-frame, not generic template)
- After final MP4 render, `extractPosterFrame(videoUrl)` in `lib/compositor.ts` extracts a representative frame via the FFmpeg API and stores the CDN URL in `Reel.thumbnailUrl`.
- WORKING FFmpeg command: `-i {{in_1}} -vf "thumbnail=100,scale=540:-2" -frames:v 1 -q:v 3 {{out_1}}`. IMPORTANT: `-ss` seeking (input OR output) FAILS on the Abacus FFmpeg API ("Output file out_1 not found") â€” use the `thumbnail` filter instead, double-quoted in the command string.
- API precedence: `app/api/reels/[id]/route.ts` and `app/api/reels/route.ts` prefer the real `thumbnailUrl` over the generic style poster via `isPlaceholderUrl()`.
- Dashboard "My Reels" cards + library cards render `reel.thumbnailUrl` as an `aspect-[9/16]` `<img>` with a Film-icon fallback.
- Composited videoUrl is a permanent `cdn.abacus.ai/videos/...mp4`; extracted thumbnails are `cdn.abacus.ai/images/...jpg`.

## Voice speed (native param, NOT FFmpeg)
- Speed presets are ElevenLabs' NATIVE `voice_settings.speed` (0.85 slow / 1.0 / 1.15 fast). Proven: same text â†’ 78KB@0.85x, 66KB@1.0x, 49KB@1.15x. NO atempo/FFmpeg.
- Dashboard sends `voice` as `${voiceId}@${speed}` (just `voiceId` when normal). `lib/providers/elevenlabs-voice.ts` `speedFromPreset()` â†’ `speedValue()`. `lib/reel-assets.ts` strips `@speed`.
- Script word budget scales with speed: `scriptPlanForDuration(targetDuration, speed)` in llm-script.ts multiplies wordBudget by speed factor. `ScriptInput.voicePreset` carries `reel.voice`; pipeline passes it at generation-pipeline.ts ~line 232.
