

## P0 FIX — Duration guarantee (requested length now enforced end-to-end)
**Root cause of the 21s-for-30s bug:** the duration picker was informational only — requested duration was never passed into or enforced by the render pipeline. Output length = raw voiceover narration length (whisper word timings); LLM script was fixed-size (~21s) regardless of target; no pad-to-target, no validation gate.

**Fixes shipped:**
- `lib/providers/types.ts`: `ScriptInput.targetDuration?`.
- `lib/providers/llm-script.ts`: `scriptPlanForDuration(target)` → word budget = max(16, round((target-2.5)/0.55)), scenes = max(4, round(budget/7)), lines = max(3, scenes-1). Prompt enforces explicit WORD BUDGET + exact line count. SAFETY NET: after parsing, drops trailing affirmation scenes (keep hook+1, min 4) while total spoken words > 1.15×budget, so narration never overruns.
- `lib/generation-pipeline.ts`: `resolveTargetDuration()` (clamp 10-60, default 25); `enforceDurationTarget(durations, speechFloors, target)` pads final scene when under target / shrinks padding (floored at speech spans) when over; XFADE=0.6. VALIDATION GATE after composite: durationDelta = finalDuration - target; if !preview && |delta|>1 → refundCoins(reason='duration_check_failed'), reel status='failed' coinCost=0, job error 'Render failed duration check — credits refunded, please retry.', [ALERT][duration-fail] log. Success scenesJson meta now has durationMet/targetDuration/durationDelta.
- `app/api/reels/generate/route.ts`: reads targetLength/targetDuration ∈ {15,20,25,30} (default 25) → pipelineOpts.
- `app/dashboard/page.tsx`: length picker (15/20/25/30 gold buttons, Clock icon, "guaranteed ±1s or credits refunded"); applyPreset sets it from preset.defaultLength.
- `lib/brand-presets.ts` + `lib/brand-presets-server.ts`: REEL_LENGTHS/LENGTHS = [15,20,25,30].
- `app/api/admin/margins/route.ts` + `app/admin/margins/page.tsx`: "Duration Accuracy (last 7 days)" widget = % of reels within ±1s per model tier, red/alert if <95%; "Duration" column in reel table (green "yes 30s" / red "🚩 no +Δs"); meta exposes durationMet/targetDuration/durationDelta.

**Compositor confirmed** pads final scene correctly for both stills (zoompan held+slow zoom) and motion clips (tpad clone + trim); captions use absolute whisper timings so extending scene video durations does not drift subtitles.
