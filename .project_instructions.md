## Reel Generation Pipeline (rebuilt for premium/viral quality)

The generation pipeline (`lib/generation-pipeline.ts`) now produces REAL composited vertical MP4s (1080x1920, h264+aac) via the Abacus FFmpeg API, replacing the old mock/stock-loop flow. Flow: script -> images -> voice -> music -> composite.

### Provider architecture (`lib/providers/`)
- `index.ts` exports `getScriptProvider` (LLMScriptProvider, model env SCRIPT_MODEL default gpt-5.4), `getImageProvider` (AbacusImageProvider, model env IMAGE_MODEL default gemini-3.1-pro, 9:16), `getVoiceProvider` (ElevenLabsVoiceProvider), `getMusicProvider` (LibraryMusicProvider).
- `fallback.ts`: `buildTemplateScript()` (curated affirmation templates w/ per-scene cinematic image prompts + timings) and `fallbackSceneImages()` (rehosts bundled /public/styles/*.jpg to public S3, rotated across scenes).
- Every step is wrapped in try/catch with graceful fallback so reels are ALWAYS produced.

### Coin / pricing model (IMPLEMENTED)
- Tiers: FREE (no paid APIs — demo gallery + ONE watermarked preview from cached/sample assets, lifetime cap 1), PRO $19.99/mo → 30 coins (static only), PREMIUM $49.99/mo → 60 coins (motion enabled).
- COIN_COST: static reel = 1 coin, motion reel = 4 coins (`lib/pricing.ts`).
- COIN_BUNDLES (one-time, Stripe, stack on subscription, valid 12 months): starter 10/$7.99, creator 25/$16.99 (popular), pro-pack 60/$34.99, studio 150/$74.99.
- Rules: coins gate VOLUME within a tier; features gated by TIER. Motion stays Premium-only even if a Pro user buys coins. Subscription coins reset monthly (UsageCounter window); bundle coins valid 12 months.
- `lib/quota.ts`: `getCoinBalance(userId)`, `checkGeneration(userId, motion)` (enforces inactive/free/motion_locked/insufficient_coins), `consumeCoins(userId, amount)` (subscription monthly first, then bundle FIFO non-expired). `checkQuota` kept as legacy shim returning reelsUsed=subscriptionUsed, reelsCap=subscriptionCoins+bundleCoins, bonusReels=bundleCoins, plus coinsAvailable/subscriptionRemaining/bundleCoins/motionEnabled.
- Schema: Reel.motion (Bool), Reel.coinCost (Int default 1). CoinPurchase.coinsRemaining (Int) + expiresAt (DateTime?). Webhook sets coinsRemaining + 12mo expiry on coin_purchase completion.
- generate route: accepts `motion` flag; free → preview pipeline mode (no paid APIs), no coin consume; paid → checkGeneration + consumeCoins up-front, sets reel.motion+coinCost.

### Compositor (`lib/compositor.ts`) — VALIDATED end-to-end
- FFmpeg API: createRunFfmpegCommandRequest / getRunFfmpegCommandStatus, deployment_token=ABACUSAI_API_KEY.
- Per-scene Ken Burns (scale 2160x3840 then zoompan to 1080x1920, alternating zoom in/out), xfade crossfades (0.6s cycling transitions), burns ASS karaoke subtitles, mixes voice(1.0)+music(0.28) or music-only(0.6) or voice-only or -an.
- input_files keys in_1.., in_music, in_voice, in_ass. ASS passed as input, referenced via subtitles={{in_ass}}. filter_complex MUST be double-quoted. vcpu_count 16, max 600s.
- Output is a permanent cdn.abacus.ai/videos/*.mp4 URL.

### Player (`components/reel-player.tsx`)
New `composited` prop. When true (videoUrl is cdn.abacus.ai composite): play video WITH baked-in audio (mute toggle controls video), NO overlay captions/music/watermark (all burned in). When false (legacy /showcase stock): muted video loop + separate music element + live caption overlay + watermark overlay. Reel page derives isComposited from scenesJson.composited.
